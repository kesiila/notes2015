使用流程

  • 查找活动
  • 收藏活动
  • 选活动
  • 选模板
  • 发邮件
选择我的产品
按行业和产品筛选，选择我的产品
关键词搜索
支持站内搜索引擎直接搜索多达600多万的活动数据库
只选有联系人和邮箱的活动
勾选联系人和邮箱前的复选框，就只显示有联系人和活动邮箱的活动
一键免费收藏
点击收藏按钮，便将公共区活动收藏到我的活动
公共区域快捷邮件
在公共区看到有兴趣的活动，可以免费给他发送一封开发信
多选活动
勾选左边的复选框，可以选择多个活动，然后发邮件，或者分享
一键全发
一键给某个活动分类里所有的活动发送邮件
选择模板
点击右边模板名称，就可以选中该模板
默认模板
如果将某个模板设置为默认模板，发邮件时就自动调用这个模板
查看发送结果
在我的邮件里，查看邮件是否发送，以及是否被活动打开
查看历史记录
也可以在查看活动详细信息时，点击右边的来往邮件按钮，查看给活动发送的所有的邮件
